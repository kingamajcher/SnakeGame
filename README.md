# Snake Game

A classic Snake game implementation in C#.

## Description

This is a simple Snake game where players control a snake to eat food and grow longer while avoiding collisions with walls and itself. The game features smooth controls and progressive difficulty as the snake grows.

## Features

- Classic snake gameplay mechanics
- Simple and intuitive controls
- Progressive difficulty as the snake grows
- Score tracking
- Game over detection

## Requirements

- .NET
- C# compatible environment
